A simple game in MakeCode Arcade. Help the ITHS Duck fight bugs!

This is a group coding effort conducted at a hackathon at ITHS.

> Open this page at [https://araneida.github.io/duck-debugger/](https://araneida.github.io/duck-debugger/)

## Use as Extension

This repository can be added as an **extension** in MakeCode.

* open [https://arcade.makecode.com/](https://arcade.makecode.com/)
* click on **New Project**
* click on **Extensions** under the gearwheel menu
* search for **https://github.com/araneida/duck-debugger** and import

## Edit this project

To edit this repository in MakeCode.

* open [https://arcade.makecode.com/](https://arcade.makecode.com/)
* click on **Import** then click on **Import URL**
* paste **https://github.com/araneida/duck-debugger** and click import
